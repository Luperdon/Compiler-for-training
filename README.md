**Название:** Разработка лексического и анализатора для конструкции вида `идентификатор = "{:f}".format(число);`

**Цель работы:** Изучение принципов построения лексических анализаторов, разработка конечного автомата для распознавания лексем, реализация анализатора для конкретного языкового конструкта языка Python.

## Постановка задачи
Разработать лексический анализатор для конструкции языка Python, представляющей собой присваивание форматированной строки с последующим вызовом метода `format()`:

идентификатор = "{:f}".format(число);

**Допустимые лексемы:**

| Код | Тип лексемы | Описание |
|-----|-------------|----------|
| 1 | Идентификатор | Имя переменной (буквы, цифры, _, начинается с буквы) |
| 2 | Ключевое слово | `format` |
| 3 | Символ в строке | Отдельные символы внутри кавычек |
| 4 | Оператор присваивания | `=` |
| 5 | Кавычка | `"` |
| 7 | Минус | `-` |
| 8 | Плюс | `+` |
| 9 | Точка | `.` |
| 12 | Целое число | Последовательность цифр |
| 13 | Вещественное число | Число с десятичной точкой |
| 14 | Отрицательное вещественное | Вещественное число с отрицательной экспонентой |
| 15 | Положительное вещественное | Вещественное число с положительной экспонентой |
| 16 | Открывающая скобка | `(` |
| 17 | Закрывающая скобка | `)` |
| 18 | Точка с запятой | `;` |
| 19 | Ошибка | Нераспознанная лексема |
| 21 | Открывающая фигурная скобка | `{` |
| 22 | Двоеточие | `:` |
| 23 | Буква f | `f` |
| 24 | Закрывающая фигурная скобка | `}` |

**Примеры корректных входных строк:**

1. `float_format = "{:f}".format(3.234e+4);`
2. `result = "{:f}".format(-123.456);`
3. `temp = "{:f}".format(0.001e-5);`
